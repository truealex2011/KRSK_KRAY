# Красноярский край - Веб-проект
# Требования для запуска

## СИСТЕМНЫЕ ТРЕБОВАНИЯ

Веб-браузер (один из):
- Google Chrome 90+
- Mozilla Firefox 88+
- Safari 14+
- Microsoft Edge 90+
- Opera 76+

## ЗАПУСК ПРОЕКТА

Вариант 1: БЕЗ СЕРВЕРА (рекомендуется)
- Просто откройте index.html в браузере
- Работает сразу, без установки

Вариант 2: С ЛОКАЛЬНЫМ СЕРВЕРОМ (опционально)

Python 3.x:
  python -m http.server 8000
  Откройте: http://localhost:8000

Node.js 14+:
  npx http-server -p 8000
  Откройте: http://localhost:8000

PHP 7+:
  php -S localhost:8000
  Откройте: http://localhost:8000

## ВНЕШНИЕ БИБЛИОТЕКИ (загружаются автоматически через CDN)

- Leaflet.js 1.9.4 (интерактивные карты)
- Google Fonts: Inter, Playfair Display

Интернет нужен только для загрузки шрифтов и карт.
Остальное работает офлайн.

## РАЗМЕР ПРОЕКТА

- Изображения: ~50-100 МБ
- Общий размер: ~100 МБ
